# Skill Acquisition & Adult Learning — Distilled Cheat Sheet (Round 4)

ของ Aree เพื่อตอบเมื่อ Toey เรียน skill ใหม่ใด ๆ ก็ตาม — ไม่จำกัดเฉพาะว่ายน้ำ. Foundational reference สำหรับทุกหัวข้อในอนาคต

---

## 🧭 Mental model

**คำถามที่ผิด**: "Toey ต้องฝึกกี่ชั่วโมง?"
**คำถามที่ถูก**: "Toey อยู่ stage ไหน + เปลี่ยน practice/feedback ยังไงให้แมตช์ stage นั้น"

3 stages ของ Fitts-Posner:
1. **Cognitive** (เริ่มต้น) — เข้าใจ task, ทำพังเยอะ, ต้องคิดทุก rep
2. **Associative** (กลาง) — ทำได้ stable ขึ้น, refining
3. **Autonomous** (ปลาย) — automatic, low cognitive load

Strategy ต่างกันแต่ละ stage. ห้าม apply random practice กับ stage 1 (จะแย่). ห้ามใช้ block practice ตลอดไป (จะ plateau).

## ⚡ The 80/20 (Round 4 — skill acquisition)

### 1. Block early, randomize later
- **Early stage**: blocked practice (เน้นทักษะเดียวซ้ำ ๆ) — เร็วในการ encode
- **Past basic competence**: random/interleaved practice — retention ดีกว่า
- **Catch**: random practice **รู้สึกแย่กว่า** ระหว่างฝึก แต่จำได้ดีกว่าวันถัดไป
- **Rule**: ถ้า accuracy in-session ดี = ผิดทาง (อาจง่ายเกิน). ถ้าวันถัดไปยังทำได้ = ถูกทาง

### 2. External focus > Internal focus (d=0.5-0.8)
- **External**: เป้าหมายในสิ่งแวดล้อม ("push the wall away", "throw the ball into the bucket")
- **Internal**: ส่วนของร่างกาย ("extend your hips", "rotate your shoulder")
- **External ชนะเสมอ** — Wulf research ใน 100+ studies, ทุกประเภท skill, ทุก expertise level
- **Translation**: เลิก cue แบบ "rotate hips", ใช้ "feel water push backward"

### 3. Faded feedback (guidance hypothesis)
- High-frequency feedback **improves practice but degrades retention** — paradox สำคัญ
- **Early**: feedback ทุก rep หรือเกือบทุก
- **Mid**: feedback ทุก 3-5 reps, summary feedback
- **Late**: feedback on-demand เท่านั้น
- **Toey จะรู้สึกว่าทำได้แย่ลงเมื่อ feedback น้อยลง** — Aree ต้อง pre-empt ว่านี่คือ desirable difficulty

### 4. Desirable difficulties (Bjork)
- **Spacing**: 3 sessions × 30 min spread > 1 session × 90 min (สำหรับ retention)
- **Interleaving**: practice A-B-C-A-B-C > AAA-BBB-CCC
- **Retrieval practice**: test yourself > re-read/re-watch
- **Generation**: solve before see solution > solution-first
- ทุกอย่าง **feel slow, learn fast**

### 5. Mental practice / imagery (d≈0.4)
- ทำงานเฉพาะเมื่อ **มี correct motor template แล้ว**
- Pure imagery บนทักษะใหม่ → rehearses errors
- **Best use**: ระหว่าง warmup, ก่อน race, ในวันพัก
- Vividness + first-person perspective + multi-sensory > flat third-person

### 6. The 10,000 hours rule = folk wisdom
- Macnamara meta-analysis: practice explains **18-26% of skill variance** (sport, music)
- Talent (genetics, early exposure) + practice quality + age of starting + practice methodology ต่างหากที่อธิบายส่วนใหญ่
- **Don't quote 10K hours rule** — quote "deliberate practice + matching methodology"

### 7. Constraints-led approach (Newell)
- Manipulate **environment** เพื่อ force pattern, ไม่ต้อง verbal-instruct
- Example swim: ใช้ pull buoy → เลิกคิดเรื่องขา, focus stroke; ใส่ paddles → cue catch feel
- **Coaching minimum, environment maximum** — adults overthink with verbal cues

## ⚡ The 80/20 (Round 4 — adult learner specifics)

### 1. Plasticity declines gradually, not abruptly
- ไม่มี cliff ที่ 25 หรือ 30 — graduated decline
- Motor skills: **0-3% per decade** typical (depends on system)
- Language: closest to "critical period" but permeable; adults can reach native-like with sufficient time + immersion

### 2. Adult ADVANTAGES (มักถูกมองข้าม)
- **Metacognition** — เข้าใจตนเอง, เลือก strategy ได้ดีกว่าเด็ก
- **Focused practice** — ไม่ฟุ้งซ่าน, ตั้งใจกว่า
- **Life context** — เห็นภาพรวม, รู้ว่าทำไปทำไม
- **Identity stability** — ไม่ติด ego เรื่อง "I'm bad at this"
- **Result**: per practice hour, motivated adult มัก **out-learn** child

### 3. Procedural >> declarative สำหรับ age preservation
- **Procedural** (basal ganglia, cerebellum): ทักษะกาย, automatic skills — **largely preserved** กับ age
- **Declarative** (hippocampus, neocortex): ข้อเท็จจริง, คำพูด — **decays first**
- แปลว่า: adult slow start ใน physical skill = declarative bottleneck (เข้าใจ instruction), ไม่ใช่ procedural
- เมื่อ skill automatize แล้ว age effects ลดลง

### 4. BDNF priming via exercise
- 20-40 min light cardio ก่อน skill practice → potentiate consolidation (animal-strong, human-suggestive)
- Mechanism: BDNF release → synaptic plasticity priming
- **Practical**: schedule learning blocks ใน 1-2 hr post-cardio window

### 5. Sleep matters MORE for adults
- Memory consolidation efficiency drops with age
- Skill plateau ที่อาจดูเหมือน "can't learn anymore" → sometimes is "can't consolidate without proper sleep"
- เกี่ยวข้องกับ Recovery hub: Round 4 doc 01 (sleep)

### 6. Identity beats hours (Dweck mindset)
- "I'm a person who learns swim mechanics" trajectory > "I need to grind hours" trajectory
- Expectancy effects: adults who expect to learn slowly → learn slowly
- Adults who set learning identity → match or exceed younger learners

### 7. Common adult traps
- **"I'm too old"** — self-fulfilling prophecy
- **All-or-nothing practice** — busy adult misses week → quits, instead of resume
- **Perfectionism** — feels worse than child; adults need permission to be cognitive-stage learner
- **Comparison to past self** — "I used to be able to..." → bad reference frame; compare to today vs yesterday

## 🧮 Application framework (Aree's decision rules)

```
When Toey starts a new skill:
1. Identify stage (cognitive / associative / autonomous)
2. Match practice variability:
   - Cognitive: blocked (50-90% same skill)
   - Associative: mixed (50-70% varied)
   - Autonomous: random/interleaved
3. Match feedback frequency:
   - Cognitive: every rep or every 2-3 reps
   - Associative: faded (every 5-10 reps + summary)
   - Autonomous: on-demand only
4. Cue with external focus, not internal
5. Add desirable difficulties at every stage (spacing minimum)
6. Schedule learning post-light-cardio when possible
7. Protect sleep around learning days
8. Set expectancy frame: "your adult brain has advantages, not just disadvantages"
```

```
When Toey plateaus:
1. Audit sleep first (Round 4 recovery hub)
2. Audit practice variability — too blocked?
3. Audit feedback — getting too much/wrong type?
4. Audit external/internal focus — slipping into internal?
5. Increase desirable difficulty (more interleaving)
6. Constraints-led intervention (change environment)
7. If still stuck: consolidation may need time; back off and revisit
```
