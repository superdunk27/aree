## 🚨 Paradigm Note — มี 2 แนวทางที่ขัดกัน

ปรัชญาด้านล่าง = **traditional sprint orthodoxy** (Auburn/USA/Australian default). แต่มี **paradigm ใหม่** ที่ Cameron McEvoy พิสูจน์แล้ว — **WR 20.88 (2026-03-20 Shenzhen)** ทำลายสถิติ Cielo 20.91 ที่ยืน ~17 ปี:

| Aspect | Traditional | McEvoy method |
|--------|-------------|---------------|
| Pool volume | 6-15 km/wk | **1-2 km/wk** |
| Per session | 4-8 km | **300-500m** |
| Sessions/wk | 6-10 | **6** (แต่สั้น) |
| Cross-training (เก่า) | Plyo + lifting | climbing + lifting |
| Cross-training (ใหม่ post-Paris) | — | **Strength + Mobility** (Matt Smith UK, สาย Poliquin) |
| Intensity | Mixed | **ทุก meter = race-pace หรือเร็วกว่า** |

**Coaches**: Matt Smith (mobility + strength now), Dan Rogers (strength pre-Paris)
**Body composition**: 87 kg Paris → 94 kg WR (intentional gain), 203 cm armspan
**Squat**: เลิก (ankle stiffness) → trap bar deadlift จาก พื้น
**Bench press**: เริ่มครั้งแรกในชีวิตปี 2025 (เคยหลีกเลี่ยง — ตอนนี้ converted)
**Junior swimmers**: Akira, Katu ใช้วิธีเดียวกัน → movement ไม่ใช่ outlier

**เมื่อใช้ McEvoy method**: ผู้ที่ **มี foundation 5+ ปี + expert coach + overtrained จาก high volume**
**เมื่อใช้ traditional**: เริ่มต้น/พัฒนาเทคนิค/ขาด strength base

## ⚡ The 80/20 — สิ่งที่สำคัญที่สุด 8 อย่าง

### 1. Underwater dolphin kick = "5th stroke"
- FINA max 15m หลัง start (และทุก turn ใน SCM)
- เป็นช่วงที่เร็วที่สุดของ race (ใน Dressel: ~2.5 m/s vs ~2.0 m/s on surface)
- Train: vertical kicks, fly kick on back/side, hypoxic underwater sets
- Threshold: ถ้า underwater speed > surface speed → ใช้ใกล้ 15m เต็ม. ถ้าตรงข้าม → break out เร็วกว่า

### 2. Posterior chain > anterior
- กล้ามเนื้อขับ sprint = glutes, hamstrings, erector, lats
- 30% ของ dryland ต้องเป็น posterior (deadlift variants, RDL, hip thrust, pullups, lat pulldowns)
- แก้ปัญหา quad-dominance ที่นักว่ายน้ำสะสมจาก kick

### 3. Plyometrics ขยับเข็ม 2-3% ใน 6-8 สัปดาห์
- non-negotiable สำหรับ sprinter
- 2-3x/wk off-season → 1x/wk in-season
- box jumps, broad jumps, depth jumps, MB throws
- = หลายเดือนของ pool work

### 4. Strength training: heavy + low rep + power-focused
- 3-6 reps @ 85-90% 1RM
- core lifts: squat, deadlift, bench, OHP, pullup
- เป้าหมาย power & RFD, ไม่ใช่ hypertrophy
- "size doesn't swim — force does"

### 5. Less yardage, more intensity (สำหรับ sprinter เท่านั้น)
- 6-8k/wk ที่ race-pace > 12-15k/wk junk yardage
- USRPT (Rushall) เป็นกรณีสุดโต่ง — ใช้แบบ hybrid ก็ได้
- ห้าม "junk yardage" — ทุก meter ต้องมีจุดประสงค์
- 2-3 high-intensity sessions/wk เท่านั้น (CNS recovery!)

### 6. Stroke length first, then rate
- DPS (distance per stroke) ก่อน → stroke rate ตามมา
- Elite: 50-65 spm. ระวังการเร่ง rate โดยไม่มี base
- ปัจจัย DPS: high catch (EVF), ankle flexibility, core stability, body line

### 7. Block start = 30% ของเวลา
- track start เร็วกว่า grab start สำหรับคนส่วนใหญ่
- reaction time goal: <0.65s (sub-elite), <0.60s (elite)
- improve via plyo + reactive drills
- 0.05s ที่ block = ~0.25s ใน finish time (เพราะ momentum carry)

### 8. Breathing: 0-2 breaths max
- Elite: 0 (Dressel ใน 50m), บางคน 1-2
- Sub-elite: อย่าฝืน 0 ถ้า aerobic ยังไม่พร้อม → จะ slow down ที่ 35m+
- ฝึกค่อยๆ ลด breath count ใน race-pace sets
- Side preference: ฝึกทั้ง 2 ข้างเพื่อ flexibility

## 🔬 Energy Systems Cheat Sheet

| ระบบ | % ของ 50m effort | Trains via |
|------|-----------------|------------|
| ATP-PCr | ~50-60% | All-out efforts ≤10s, แล้ว rest 2-3 min |
| Anaerobic glycolysis | ~30-40% | 25-50m repeats with full rest |
| Aerobic | <10% | base work, easy swimming, cross-training |

**Lactate after 50m**: typically 10-15 mmol/L. Cool-down protocol สำคัญสำหรับ next race ใน meet เดียวกัน

## 💊 Supplements with Evidence (for 50m specifically)

| Supplement | Dose | Evidence | Notes |
|-----------|------|----------|-------|
| **Creatine monohydrate** | 3-5 g/d (no loading needed) | Strong (sprint power) | safe long-term, may +1-2 kg water weight |
| **Caffeine** | 3-6 mg/kg, 30-60 min pre-race | Strong | individual tolerance varies, test in training first |
| **Beta-alanine** | 4-6 g/d for 4+ weeks | Moderate | tingling = expected. Effect on <60s events debated |
| **Sodium bicarbonate** | 0.2-0.3 g/kg, 60-90 min pre | Moderate | GI distress common, requires testing |
| **Sodium nitrate (beetroot)** | 6-8 mmol nitrate, 2-3 hr pre | Weak for sprint | better for endurance |

**ที่ Aree ไม่แนะนำ**: pre-workouts ที่มีหลายตัวรวม (dose ของแต่ละตัวมักต่ำเกิน, นักว่ายน้ำมัก fail drug tests จาก contamination)

## 🍽️ Nutrition Daily

- **Protein**: 1.6-2.2 g/kg/d
- **Carbs**: 5-7 g/kg/d (heavier weeks: up to 10)
- **Fat**: ~1 g/kg/d minimum
- **Pre-race meal**: 3-4 hr before, balanced (carbs + protein + low fat/fiber)
- **30-60 min before**: fast carbs (banana, dates, sport drink) + caffeine
- **Recovery**: 0.3 g/kg protein + carbs ≤30 min post-session

**Energy availability**: ห้าม chronic underfueling — RED-S syndrome ทำลาย sprint power

## 🛡️ Shoulder Prehab (Non-Negotiable)

40-91% incidence ของ shoulder injury ใน competitive swimmers. ทำ daily หรืออย่างน้อย 3-4x/wk:

- External rotation (cable/band), 2x10-15
- Y-T-W on bench (light DBs), 2x8-10 each
- Scapular pulldowns, 2x12
- Wall slides, 2x10
- Sleeper stretch (gentle), hold 30s

**สังเกต early warning**: ปวดหลังว่าย, แขนชา → หยุดและประเมิน ไม่ใช่ "ฝึกผ่าน"

## 📅 Sample Weekly Microcycle (Off-Season Sprint Block)

| Day | AM | PM | Notes |
|-----|----|----|-------|
| Mon | Pool 6k tech + race-pace 8x25 max | Dryland: heavy lower (squat, RDL, hip thrust) | High intensity day |
| Tue | Pool 4k easy + skill | Plyometrics + upper push | CNS recovery |
| Wed | Pool 6k aerobic + threshold | Rest | Active recovery |
| Thu | Pool race-pace 12x25 max + 4x50 broken | Dryland: pulls, posterior, core | High intensity day |
| Fri | Pool 4k tech + 4x100 race-pace | Plyometrics + mobility | |
| Sat | Pool sim/test set OR rest | Long mobility / sauna | |
| Sun | Rest | Rest | Full recovery |

**In-season**: ลด volume, รักษา intensity, ลด strength volume แต่ keep loads heavy

## 🚨 Red Flags / Common Mistakes

1. **เพิ่ม yardage หวังจะเก่ง** — sprinter พังจาก over-aerobic work
2. **ทำ dryland ก่อน race-pace pool** — pre-fatigue ทำลาย quality
3. **ไม่ taper** — sprinter ต้องการ 1-3 wk taper, ลด vol 40-60%
4. **ใส่ paddles ใหญ่เกิน** — shoulder injury #1 cause
5. **ฝืน 0 breath ทั้งที่ยังไม่พร้อม** — สลายไปที่ 35m
6. **Skip warmup** — risk pull/strain โดยเฉพาะคนอายุมาก
